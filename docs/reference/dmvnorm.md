# dmvnorm

Calculates the log density of a multivariate normal distribution with
mean mu and covariance matrix Sigma.

## Usage

``` r
dmvnorm(x, mu, Sigma)
```

## Arguments

- x:

  Vector: The value from to which calculate the density.

- mu:

  Vector: The mean vector

- Sigma:

  Matrix: The Covariance matrix.
