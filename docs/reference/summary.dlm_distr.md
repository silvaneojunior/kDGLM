# Summary for a kDGLM outcome

Prints a report for a dlm_distr object.

## Usage

``` r
# S3 method for class 'dlm_distr'
summary(object, ...)
```

## Arguments

- object:

  A dlm_distr object.

## Value

No return value, called to print a summary of a kDGLM outcome.
