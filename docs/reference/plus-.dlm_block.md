# +.fitted_dlm

Define add operator for class dlm_block. This method is wrapper for the
block_superpos function.

## Usage

``` r
# S3 method for class 'dlm_block'
e1 + e2
```

## Arguments

- e1:

  A dlm_block.

- e2:

  A dlm_block.

## Value

The combined blocks as a dlm_block.
