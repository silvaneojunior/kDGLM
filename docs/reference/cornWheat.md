# Corn and wheat prices from 1986 to 2014

The to prices (in U.S. Dollars) per bushel and the log returns of corn
and wheat from 1986-01-03 to 2014-10-10. Each observation corresponds to
the price on that day, but not all days are present in this dataset.

## Usage

``` r
cornWheat
```

## Format

A data frame with 7,253 rows and 5 columns:

- date:

  The date of the observation.

- corn.price, wheat.price:

  The price (in U.S. Dollars) per bushel of corn and wheat,
  respectively.

- corn.log.return, wheat.log.return :

  The log returns for corn and wheat, respectively.

## Source
