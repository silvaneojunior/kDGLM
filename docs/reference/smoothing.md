# Auxiliary function for model smoothing

Auxiliary function for model smoothing

## Usage

``` r
smoothing(model)
```

## Arguments

- model:

  A fitted_dlm object.

## Value

A fitted_dlm object with smoothed means (mts) and covariance matrix
(Cts) for each observation.
