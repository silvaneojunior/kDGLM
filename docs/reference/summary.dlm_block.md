# Summary for a kDGLM structure

Prints a report for a dlm_block object.

## Usage

``` r
# S3 method for class 'dlm_block'
summary(object, ...)
```

## Arguments

- object:

  A dlm_block object.

## Value

No return value, called to print a summary of a kDGLM structure.
